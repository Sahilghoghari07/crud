# CRUD With MERN🚀

This repository is created to learn, practice, and master backend development using modern JavaScript.  
It focuses on building **real-world server-side applications**, understanding backend architecture, and writing clean, scalable code.

---

## 📌 About This Repository

The purpose of this repository is to:

- Strengthen **Node.js fundamentals**
- Understand backend concepts & server architecture
- Build RESTful APIs
- Work with databases

Each project inside this repository is independent feature or concept.

---

## ▶️ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sahilghoghari07/nodejs.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd project-folder-name
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Server

```bash
npm run dev
```

> Make sure to create a `.env` file and add required environment variables before running the project.
